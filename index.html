Beyond OWASP

6 Novel Rules Beyond the Standard

AgentGuard extends beyond the OWASP ASI Top 10 with proprietary detection rules designed for real-world autonomous agent threats.

🔒

MCP Supply Chain Integrity

Detects malicious MCP server manifests before connection. Validates tool signatures, checks for data exfiltration endpoints, and blocks servers with known compromise indicators.

🔄

Cross-Agent Contamination

Prevents context leakage between agent sessions. Identifies unauthorized data sharing across agent boundaries and blocks state pollution from prior invocations.

🔃

Recursive Tool Loop Detection

Identifies infinite and runaway tool call patterns. Detects cyclic dependencies, exponential retry chains, and resource exhaustion vectors before they execute.

🔑

Credential Exposure in Logs

Scans agent trace output, debug logs, and tool responses for leaked API keys, tokens, and secrets. Flags both plaintext and obfuscated credential patterns.

📝

Over-Permissioned Tool Access

Audits tool permission scopes against least-privilege principles. Flags tools granted filesystem, network, or shell access without adequate sandboxing.

🔍

Agent Decision Auditing

Traces and verifies critical agent decisions with tamper-proof audit logs. Detects anomalous decision paths, unauthorized escalation, and policy violations.
